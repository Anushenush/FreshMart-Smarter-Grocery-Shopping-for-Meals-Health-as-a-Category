# Replit No-Code Prompt: Grocery Shopping Web App

Create a modern grocery shopping web application using only Bootstrap 5 for UI and layout. Do not use Tailwind CSS.

## Theme and Style

* Use a premium purple and red mixed gradient theme across the app.
* Main colors: deep purple, violet, magenta, dark red, soft pink highlights.
* Backgrounds should use subtle dark gradients with glowing purple-red cards.
* Use glassmorphism effects, rounded corners, shadows, hover lift effects, glowing buttons, and animated gradients.
* Add smooth animations like fade-in sections, slide-up cards, hover zoom on products, animated cart icon, floating badges, and smooth transitions.
* Overall style should feel modern, futuristic, clean, premium, and mobile responsive.

## Main Purpose

Build a grocery shopping app focused mainly on these 2 core features:

1. Recipe-to-Cart One-Click Integration
2. Hyper-Personalized Health Filters

## Homepage Sections

### Hero Section

* Large heading: “Smarter Grocery Shopping for Meals and Health”
* Subheading explaining recipe-to-cart and health-based shopping.
* Add 2 CTA buttons:

  * “Explore Recipes”
  * “Shop by Health Goals”
* Add animated grocery illustrations, floating food icons, and glowing cart animation.

### Feature Cards Section

Show large Bootstrap cards for the main features:

## Core Feature 1: Recipe-to-Cart One-Click Integration

Create a dedicated section called “Cook by Recipe”.

Add:

* Search bar for meals
* Meal categories like:

  * Breakfast
  * Lunch
  * Dinner
  * Snacks
  * Healthy Meals
  * Kids Meals
  * Vegetarian
  * High Protein
  * South Indian
  * North Indian
  * International Meals

When user selects a meal, instantly show:

* Meal image
* Servings selector (1 person, 2 people, 4 people, 6 people)
* Cooking time
* Difficulty level
* Ingredients required with exact quantities
* Nutritional summary
* “Add Full Recipe to Cart” button

Example meals:

* Butter Chicken for 4 People
* Biryani for Family
* Dosa and Chutney Combo
* Paneer Butter Masala
* Chicken Fried Rice
* Pasta Alfredo
* Veg Sandwich
* Idli Sambar

When user clicks “Add Full Recipe to Cart”:

* Automatically add all ingredients with exact quantities to cart.
* Show animation where ingredients fly into the cart.
* Show success toast notification.
* Cart updates live.
